Front-End Projects

  • InfinitGem

    Made: 2024

    The Infinitgem project is an interactive web game inspired by The Useless Web where players explore a cave to find hidden gems. The game features a dynamic layout where gems are randomly placed within a virtual cave. When a player clicks on a gem, it duplicates and is randomly repositioned, increasing the challenge as the number of gems grows. The design utilizes HTML, CSS, and JavaScript, with SVG elements for the cave background and gem images. Players can reset the game by refreshing the cave, creating an engaging and re-playable experience.

  • Spirited Sips

    Made: 2024

    This project is a recipe blog post template for a website called "Spirited Sips." The design emphasizes readability and user interaction, featuring a central content area for detailed cocktail recipes, with a sidebar showcasing trending posts. The template uses a soft color palette of light blues and purples and includes sections for images, ingredients, instructions, and notes, alongside user comments. The layout is structured using CSS flexbox for responsiveness, ensuring that the content is accessible and visually appealing across different devices.

  • CSS Experiments

    Made: 2024

    This project showcases a collection of creative CSS animations and interactive elements, each demonstrating unique effects and concepts. The website features various experiments, including a Record Player Animation (a record spins and plays when the play/pause button is clicked, complete with a moving arm to simulate a real turntable), a Beating Heart( an icon that beats and changes colour when hovered over), a Spilling Mug (animation where a mug tilts and spills coffee when clicked, using keyframe animations to create the flowing liquid effect), a Bouncing Letter â€˜i'(where parts of the letter 'i' bounce when the user hovers over the box containing it), and a Blooming Flower (blooms when clicked, with petals unfolding in sequence and a changing background to simulate sunlight). These experiments highlight the potential of CSS to create engaging and dynamic web experiences. Each interaction is designed to be responsive and visually appealing, making it a great resource for learning and inspiration in web development.

  • Cocktail Generator

    Made: 2024

    This Cocktail Generator is a web-based application designed to help users create cocktails based on the ingredients they have on hand. The application allows users to select different liquors and mixers from a list of available options, and then generates cocktail recipes that can be made with those ingredients. It features an ingredient selection with a variety of liquors and mixers and an option to browse through all the recipes. The app provides a smooth user experience with form handling and dynamic content updates without page reloads. This project showcases a practical use of JavaScript and jQuery for creating an interactive and engaging web application. It's a fun tool for anyone interested in mixing drinks, whether they are experienced bartenders or just looking to try something new at home.
