  • Grooovy Tunes

    2024

    Grooovy Tunes is a full-stack web application built to simulate the online presence of a record store. It allows users to browse available vinyl records, while admin functionality enables adding, editing, and deleting items in the catalog. This solo project demonstrates end-to-end CRUD functionality, with a MongoDB-backed database, Express routing, and dynamic server-rendered views using Pug templates. I was responsible for the entire build, from database schema design to UI layout and application logic. The focus of this project was to create clean, maintainable code and practice full-stack development fundamentals.

  • Art in Context

    2024

    Art in Context is a web application that explores the relationship between visual art and history. Built using Express.js and EJS, the project draws from the Art Institute of Chicago’s public API to display artwork alongside significant historical events from the same year, using data from the API Ninjas Historical Events API. The result is a more immersive and thoughtful user experience that helps viewers see beyond the canvas and understand what was happening in the world when the piece was created.

    This project was inspired by the idea that art doesn’t exist in a vacuum — it reflects the politics, culture, and events of its time. By placing each artwork in its historical context, the application encourages users to consider how these external forces may have influenced the artist’s perspective, style, or subject matter. In the future, I plan to expand the app with features like pagination for browsing larger collections, a search function, and filtering options by artist, medium, or type. These enhancements will make the platform even more dynamic and accessible for users looking to explore art through a broader cultural lens.

  • InfinitGem

    2024

    The Infinitgem project is an interactive web game inspired by The Useless Web where players explore a cave to find hidden gems. The game features a dynamic layout where gems are randomly placed within a virtual cave. When a player clicks on a gem, it duplicates and is randomly repositioned, increasing the challenge as the number of gems grows. The design utilizes HTML, CSS, and JavaScript, with SVG elements for the cave background and gem images. Players can reset the game by refreshing the cave, creating an engaging and re-playable experience.

  • CSS Experiments

    2024

    This project showcases a collection of creative CSS animations and interactive elements, each demonstrating unique effects and concepts. The website features various experiments, including a Record Player Animation (a record spins and plays when the play/pause button is clicked, complete with a moving arm to simulate a real turntable), a Beating Heart( an icon that beats and changes colour when hovered over), a Spilling Mug (animation where a mug tilts and spills coffee when clicked, using keyframe animations to create the flowing liquid effect), a Bouncing Letter ‘i'(where parts of the letter 'i' bounce when the user hovers over the box containing it), and a Blooming Flower (blooms when clicked, with petals unfolding in sequence and a changing background to simulate sunlight). These experiments highlight the potential of CSS to create engaging and dynamic web experiences. Each interaction is designed to be responsive and visually appealing, making it a great resource for learning and inspiration in web development.

  • Cocktail Generator

    2024

    This Cocktail Generator is a web-based application designed to help users create cocktails based on the ingredients they have on hand. The application allows users to select different liquors and mixers from a list of available options, and then generates cocktail recipes that can be made with those ingredients. It features an ingredient selection with a variety of liquors and mixers and an option to browse through all the recipes. The app provides a smooth user experience with form handling and dynamic content updates without page reloads. This project showcases a practical use of JavaScript and jQuery for creating an interactive and engaging web application. It's a fun tool for anyone interested in mixing drinks, whether they are experienced bartenders or just looking to try something new at home.
